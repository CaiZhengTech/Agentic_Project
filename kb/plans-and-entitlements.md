# What each plan includes (Basic / Pro / Enterprise)

Northbeam IT Services offers three plans. This page lists exactly what's included at each
tier so you can check your own entitlements before opening a ticket, and so support can
explain clearly when a request is outside your current plan.

## Basic

Included with every Northbeam account:

- Standard support (best-effort, normal ticket queue, no response-time guarantee)
- Email client setup (IMAP/SMTP configuration assistance)
- Software installation and update help
- Password reset and account unlock

Basic does **not** include priority VPN support, API access, data export, a dedicated IP,
or custom integrations. If your ticket requests any of these, it needs an upgrade — see
"Upgrading or downgrading your plan."

## Pro

Everything in Basic, plus:

- **priority_vpn_support** — VPN tickets are triaged same-day instead of standard queue order
- **api_access** — issuance of developer API keys (see "API access and developer keys")
- **data_export** — self-service export of your account data and backups (see "Exporting
  your data and backups")

## Enterprise

Everything in Pro, plus:

- **dedicated_ip** — a static IP address reserved for your account's VPN and API traffic
- **custom_integrations** — support for connecting Northbeam to third-party systems beyond
  the standard integrations catalog (contact sales to scope)

## Important: what support can and cannot do

Support agents (including our automated triage assistant) can help you understand what your
plan includes and can point you to the upgrade path. **Support cannot grant a feature that
isn't on your plan** — for example, a Basic account cannot be given priority VPN support,
API access, or a dedicated IP without upgrading first. Any request that would require
granting a feature outside the account's current plan is routed to a human reviewer rather
than resolved automatically, so a person can confirm the plan details with you and offer the
correct upgrade path if you want one.

## When to contact support

Contact support if you're unsure which plan you're on, or if a feature you expect to have
access to isn't working — we'll check your entitlements and explain what's included. To add
a feature, see "Upgrading or downgrading your plan" or "Contacting sales and pre-sales
questions."
